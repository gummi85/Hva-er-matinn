
Hvað er í matinn? PWA

Til að prófa:
1. Afþjappa zip skránni.
2. Setja möppuna á GitHub Pages, Netlify eða annan static host.
3. Opna slóðina í síma.
4. Velja Add to Home Screen / Bæta á heimaskjá.

Ath:
- Service worker þarf venjulega https eða localhost.
- Gögn vistast local í vafranum.
- Þetta er publish-ready grunnur fyrir næsta skref.
